## 자바스크립트 입출력

- 입력 -> 연산 -> 출력

구글링

    쇼핑몰 만드는 사이드 프로젝트
    쇼핑몰 만드는 법 (x)
    low - level의 알고리즘 단위 검색 (o)

출력 값은 이미 알고 있는 값
출력을 토대로 입력 값을 일부 에상
연산을 하기 전 먼저 설계를 완료하라 ----> 구현 가능한지 검증
설계가 완료되면 코드를 작성한다
